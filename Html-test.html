  • Q: Describe the differences between the IE6, IE7, and W3C box models.

    In the W3C box model, the width of an element gives the width of the content of the box, excluding padding and border.
    In the traditional box model, the width of an element gives the width between the borders of the box, including padding and border.
    Internet Explorer in "quirks mode" includes the content, padding and borders within the specified width or height.
    This results in narrower or shorter boxes than the standard behavior.

    Adventure Question


    Describe the MVC design pattern as it applies to web servers - start from initial HTTP request and end with writing the response.

    The Model handles the state of the application. It just supplies ways to query the state, and ways to change that state.
    The View is the representation of the user interface. A view can query the model, but it is not supposed to change the state.
    User actions on the View are send to the Controller.The Controller receives user requests, and translates them into actions that the Model should take.
    Then it selects the appropriate View to handle the response.
    Steps :
    1) Browser sends a http request to a Dispatcher servelet which dispatches to an appropriate controller.
    2) Controller queries the model.
    3) Model returns the status to the controller.
    4) The controller then selects a view based on the request.
    5) The view then queries the state of the model.
    6) The model returns the state of the model to the view.
    7) The view returns a http response to the browser.
